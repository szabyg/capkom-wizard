Adds a symbol to a Capkom UI. Depending on the profile and the symbolset the symbol can come from different symbolsets and have different sizes. Easiest usage would be this:

<div symbolId="welcome" class="label">
    Welcome
</div>

jQuery(".label").capkomSymbol();

That causes a lookup for the welcome symbol in the defined symbolsets in the defined size.

 
jQuery.widget "Capkom.capkomSymbol"
¶

¶

By default Capkom.profile is used for getting the selected symbolset, size and to be informed about profile changes.

 
        profile: Capkom.profile
¶

By default Capkom.symbolSets is used to get the symbolsets

 
        symbolSets: Capkom.symbolSets
¶

If no symbolId is defined on a label use default

 
        symbolId: "default"
¶

For testing only, you can put an additional prefix before the symbol uri. This makes sense if the symbolsets are defined relative to the page uri.

 
        uriPrefix: ""
