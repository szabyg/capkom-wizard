CapkomSymbol widget

This widget is a play for detecting the ideal symbol size for a person. It measures coordination skills of clicking on a symbol. The symbol size is big in the beginning and becomes smaller and smaller after each level. The sizes depend on the screen size and the speed of the player. The game begins with a size that's at least maxSize specified in the options and ends at the latest when the measured symbol size became smaller then the minSize. For further info see the documented source code or github
